
LEEME PARA USAR ESTA APLICACION COMO BASE PARA OTRAS EN ANDROID ESTUDIO

1. COPIAR ESTE DIRECTORIO COMPLETO CON OTRO NOMBRE
2. CON UN EDITOR DE TEXTO, EDITAR app/build.gradle EN LA NUEVA COPIA, CAMBIAR EL NOMBRE DEL PAQUETE
   applicationId "es.trapasoft.android.blank" POR EL NUEVO
3. EDITAR app\src\main\AndroidManifest.xml, CAMBIAR package="es.trapasoft.android.blank" POR EL MISMO NOMBRE
   DE PAQUETE QUE HEMOS PUESTO EN (2)
4. BORRAR NombreProyectoAnterior.iml Y app\app.iml
5. IMPORTAR EL NUEVO PROYECTO EN ANDROID STUDIO
6. EL PACKAGE DENTRO DE app\java SE SEGUIRA LLAMANDO COMO EL PROYECTO ANTERIOR: Refactor/Rename AL NUEVO NOMBRE
7. Build/Clean Project + Build/Rebuild Project
